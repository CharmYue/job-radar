# 🎬 视频脚本

两个版本:**90 秒短视频**(小红书 / 即刻 / B 站封面)+ **3-5 分钟完整 demo**(YouTube / B 站长视频)。

## 工具

- 录屏:macOS `Cmd + Shift + 5` 录区域 / Camtasia / OBS / [ScreenStudio](https://www.screen.studio/)(强烈推荐 — 自动鼠标平滑 + 缩放)
- 配音:用自己声音或 [11labs](https://elevenlabs.io/) AI 配音
- 字幕:Final Cut Pro / [CapCut](https://www.capcut.com/) / [Descript](https://www.descript.com/)(自动转字幕)
- BGM:[YouTube Audio Library](https://www.youtube.com/audiolibrary)(免版权)

## 90 秒短版 — 钩子优先

### 0-5s — **痛点钩**

**画面**:Boss直聘网页快速滑动几百个岗位(2 倍速),光标焦虑地选来选去,最后停下来叹气。

**字幕**:
> 一天 300 个岗位,人肉筛 3 小时还容易漏。

### 5-15s — **方案钩**

**画面**:剪辑回扩展 popup,镜头特写 🚀 跑一轮 按钮,按下去。

**字幕**:
> 一个 Chrome 扩展,AI 帮你扫一遍。

### 15-35s — **核心 demo:画像 → 搜索**

**画面**:
- popup 画像 tab,镜头扫过 "你的需求 / 简历 / 滑块 / 6 维偏好星级"(4-5 秒)
- 切搜索 tab,关键词 chips 弹出来,城市勾上,"将抓 540 个岗位"(3 秒)
- 切运行 tab,点 🚀,阶段条亮起来(2 秒)

**字幕**(分段):
> 1. 简历 + 你想要什么 — 告诉它一次
> 2. 关键词 + 城市 — 你想搜什么
> 3. 一键跑

### 35-65s — **核心 demo:跑完看结果**

**画面**:
- 阶段条 ① 采集 → ② 打分 → ③ 推送 progressively 高亮(快进效果,实际是 3 段录像剪在一起)
- 切到全屏看板,**慢镜头** 镜头从顶部 toolbar 扫到 S 级岗位的评分理由 + 担忧 + HR 绿色活跃徽章
- 镜头停在一行 S 级岗位上 1 秒

**字幕**:
> S → A → B → C 自动分档
> 评分理由 / 担忧点 / HR 是否还活跃 全告诉你

### 65-80s — **二次价值**

**画面**:
- 点 📥 CSV,Excel 打开,按 HR 活跃度筛选
- 切手机:WxPusher 微信推送来了,S 级 + A 级 Markdown 渲染

**字幕**:
> 一键导 Excel · 微信收推送 · 不投烂岗

### 80-90s — **CTA**

**画面**:GitHub README 顶部,version 6.2.1 + 仓库地址。

**字幕**:
> 开源 · 用你自己的 AI key · 不上传数据
>
> github.com/CharmYue/job-radar

## 3-5 分钟完整版

### 0-15s — 开场

跟短版 0-15s 一样,但留出 5 秒展示"5 分钟跑起来"标题。

### 15s-1m — 装扩展 + 配画像

实时录:
1. clone 仓库,加载已解压扩展(15s)
2. 画像 tab 一字一字填(用准备好的样例文字,加速 2-3 倍),重点演示:
   - 滑块拖动月薪 / 年包
   - 6 维偏好星级一颗一颗点
   - 选 DeepSeek + 粘贴 API key + 点 🧪 测试模型连通(成功提示)

### 1m-1m30s — 配搜索

- 输入关键词 chip,enter
- 勾城市
- 高级里改 "每词最多抓 60"
- 显示 "将抓 540 个岗位"

### 1m30s-2m30s — 跑流水线

实时录(可以加速):
- 点 🚀,新窗口打开 Boss 列表页(最小化)
- popup 回到运行 tab,看 log 滚动
- 阶段条切换:采集中 → 打分中 → 推送中

**配音**:解释 "alarm-driven 状态机 — SW 死了也能续跑"

### 2m30s-4m — 看结果

- popup 卡片视图,展开一条 S 级 — 念出评分理由 + 担忧
- 切表格视图,演示 HR 颜色
- 点 🖥 全屏 — 演示排序 + 全文搜索
- 操作:标记已投 + 屏蔽公司 + 📥 CSV
- 切手机演示 WxPusher 推送

### 4m-5m — 技术细节(可选,给极客观众)

- 打开 SW DevTools,展示 IndexedDB 里的 jobs store + indexes
- chrome.alarms 列表
- 提一句 "MV3 SW 续跑 / IDB 抗 race / 指纹去重 / 7 个 AI 模型 / 不发请求只观察"

### 5m — Outro

- GitHub 地址
- "MIT 开源 · 欢迎贡献"

## 录制小贴士

- 录之前**清干净测试数据**(避免暴露真实信息):
  - chrome.storage.local 清掉 profile / apiConfig
  - IDB 清掉 jobs(运行 tab 点 "清数据池")
  - 重新装一份 demo 数据(可以用 fake API key + Boss 上跑 2-3 个关键词)
- popup 截屏前**关掉自动保存徽章**(`✓ 已自动保存` 闪烁会显得很乱)— 或者就接受
- **打码**:每段录像都过一遍,真实公司名 / 手机号 / 邮箱用模糊
- **节奏**:90 秒版本镜头快(2 秒一个画面),完整版可以慢一点
- **配音**:**说人话**,别背 README。例 "你的需求 — 一句话告诉它" 而不是 "在画像 tab 配置 summary 字段"
